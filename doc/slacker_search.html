all/3

all(Token::string(), Query::string(), Options::list()) -> http_response()

Search for messages and files matching a query.

Options can be: sort: return matches sorted by either score or timestamp (default: score) sort_dir: asc or desc (default: desc) highlight: pass a value of 1 to enable query highlight markers count: number of items to return per page (default: 20) page: page number of results to return (default: 1)
