info(Token::string(), File::string(), Options::list()) -> http_response()

Options can be: count: number of items to return per page (default: 100) page: page number of results to return (default: 1)

list/2

list(Token::string(), Options::list()) -> http_response()

List and filter team files.

Options can be: user: filter files created by a single user channel: filter files appearing in a specific channel ts_from: filter files created after this timestamp (default: now) ts_to: filter files created before this timestamp (default: now) types: filter files by type (default: all) count: number of items to return per page (default: 100) page: page number of results to return (default: 1)
